# UI Spec Viewer

Markdown形式で画面設計書を表示するための拡張機能。
`---` と `--NN` のスニペットで左右に分割表示する。

## 機能（MVP）

- 左右分割表示（デフォルト50:50）
- `---` から `--NN` の間を左、`--NN` から `---` の間を右として表示
- `--NN` の `NN` で左:右比率を指定（10-90）
- 複数ブロックを連続表示
- Markdown編集中と保存時のホットリロード
- 区切り不足時の警告表示と自動補正

## 使い方

1. このディレクトリで依存をインストール
2. TypeScriptをビルド
3. VS Codeでこのフォルダを開いて拡張開発実行（F5）
4. Markdownファイルを開き、コマンドパレットから `UI Spec Viewer: Open Split View` を実行

```bash
npm install
npm run compile
```

## Markdown記述ルール

1つの分割ペアは次の構文で定義する。

- `---` から `--NN` まで: 左側
- `--NN` から `---` まで: 右側

```md
---
## 左側: 画面
![ログイン画面](./assets/login.png)

--50
## 右側: 仕様
- メールアドレス
- パスワード
---

---
## 左側: ダッシュボード
![ダッシュボード](./assets/dashboard.png)

--40
## 右側: 仕様
- KPIカードを上部に表示
- 一覧テーブルを下部に表示
---
```

分割スニペット未使用時は通常のMarkdownプレビューとして表示する。

## 注意

- 最後の右ブロックが `---` で閉じられていない場合は自動で閉じて表示する
- `--NN` の範囲外指定は 10-90 に補正する
