## 右側: ダッシュボード仕様

### 目的
ログイン後に主要KPIと最新情報を確認する。

### 要素
- KPIカード（売上、継続率、新規登録）
- 最近のアクティビティ一覧
- 通知エリア

### 操作
- KPIカードクリックで詳細ページへ遷移
- 一覧行クリックで詳細モーダルを表示
